Quick Start
================================================================================

Development (with hot reload):
  $ docker compose up
  → All services start with hot reload
  → Open dashboard: http://localhost:8005

Development (with file watching):
  $ docker compose watch
  → Automatic sync when files change
  → Faster iteration than bind mounts alone

Production:
  $ docker compose -f docker-compose.prod.yml up -d
  → Services start with restart policies
  → Resource limits enforced
  → Only dashboard exposed (port 8005)

Build only:
  $ docker compose build
  → Build all images without starting containers

View logs:
  $ docker compose logs -f dashboard
  → Stream logs from dashboard service

Clean up:
  $ docker compose down -v
  → Remove containers and volumes
